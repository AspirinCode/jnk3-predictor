How to use this website

This website is an interface which allows you to predict whether your molecules of interest inhibit the JNK3 protein (also known as MAPK10). You can either draw the molecules you are interested in, or paste an extisting list of SMILES strings for compounds of interest. After pressing the "Predict" button, you will see the probability of a given compound inhibiting the JNK3 protein, inferred by the Logistic Regression and Random Forest models, trained on two different datasets, each represented by two different descriptor sets.

Step 1. Input your molecules of interest.

First, draw one of your molecules in the JSME Editor. Then press the blue "Add molecule" button below. This will add the drawn molecule's SMILES to the SMILES input area below. Instead (or additionally), you can manually edit the SMILES input area, listing the SMILES representations for the molecules you want to check. Please, separate them with commas (,) using no spaces. Step 2. Get the predictions

After you have inputted your molecules' SMILES, press the green "Predict" button. This will send the SMILES strings from the input area to the server, process the data, and display the results as a table on the right. All in all, the server makes use of 8 machine learning models:

  • Dataset 1 (having randomly selected inactive compounds)
    • 2D RDKit Descriptors
      • Logistic Regression
      • Random Forest
    • Mordred Descriptors
      • Logistic Regression
      • Random Forest
  • Dataset 2 (having inactive compounds selected using an autoencoder)
    • 2D RDKit Descriptors
      • Logistic Regression
      • Random Forest
    • Mordred Descriptors
      • Logistic Regression
      • Random Forest

Thus, the table on the right displays the predictions obtained from all of the 8 models for each of the compounds that you inputted. The predictions can be considered a model's opinion on the probability that a given molecule inhibits the JNK3 protein.

More info on the methodology of the experiment can be found here.

Step 3. Restart the process

Once you are done analyzing the predictions, you can press the red "Clear all" button to clear the predictions' tables as well as the SMILES input area. The JSME Editor has to be cleared manually by pressing the blank rectangle next to a smiling face on the upper left of the editor window.
